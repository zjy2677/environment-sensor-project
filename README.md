# Environment Sensor Project

This project builds a environment-sensor with a DHT11 sensor, 4 digits 7 segments display and a IC 74HC595N to measure real time temperature and humidity in the surronding environment.

## Project structure

- `docs/` includes wiring diagrams and instructions 
- `docs/photos/` for hardware and setup photos.
- `docs/video/` includes a demo video that will be directly displayed below
- `src/` includes all the codes for arduino

## Hardware setup
- DHT11 3-legs temperature and humidity sensors
- 4 Digits 7 Segments display
- IC 74HC595N
- 8 220 ohms resistors
